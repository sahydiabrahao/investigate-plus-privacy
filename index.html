Investigate+ — Política de Privacidade

Última atualização: 2026-03-28

A extensão Investigate+ respeita totalmente a privacidade do usuário. Esta extensão não coleta, não armazena, não compartilha e não transmite dados pessoais para nuvem, servidores externos ou terceiros.

📌 1. Nenhuma coleta de dados

A extensão não coleta:

  • Nome, e-mail, telefone ou qualquer informação pessoal.
  • Histórico de navegação.
  • Conteúdos de sites visitados.
  • Dados de uso ou métricas.
  • Informações financeiras, sensíveis ou identificáveis.

📌 2. Arquivos acessados localmente

A extensão permite que o usuário:

  • selecione uma pasta local no computador
  • visualize arquivos locais, como PDF, imagens, vídeos e JSON
  • crie, leia e edite arquivos JSON dentro da pasta escolhida
  • abra arquivos em nova aba localmente

Nenhum arquivo é enviado para servidores.
Todo o processamento acontece localmente no dispositivo do usuário.

📌 3. Armazenamento e funcionamento local

A extensão funciona apenas com arquivos e pastas escolhidos manualmente pelo usuário no próprio dispositivo.

Informações de uso da pasta selecionada e arquivos manipulados pela extensão permanecem no ambiente local do navegador e no dispositivo do usuário. Não há sincronização com nuvem, servidores externos ou serviços de terceiros.

📌 4. Nenhum compartilhamento de dados

Como nenhum dado é coletado, nenhum dado é compartilhado com terceiros.

📌 5. Arquivos JSON criados e modificados localmente

A extensão permite que o usuário crie e edite arquivos JSON em uma pasta local escolhida manualmente.

Todo o conteúdo desses arquivos é armazenado exclusivamente no dispositivo do usuário. Esses dados não são enviados, transmitidos ou sincronizados com qualquer servidor, serviço externo ou terceiros.

O usuário tem total controle para criar, editar e excluir esses arquivos a qualquer momento.

📌 6. Controle do usuário

O usuário decide quais pastas abrir, quais arquivos visualizar e quais dados editar. A extensão não acessa conteúdo fora do que for selecionado manualmente.
